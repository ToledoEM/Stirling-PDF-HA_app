# Stirling-PDF Home Assistant Add-on Repository

A Home Assistant OS add-on repository that packages [Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF) — a locally hosted, feature-rich PDF manipulation tool with 60+ tools.

## Add-ons in this repository

- **[Stirling-PDF](stirling_pdf/)** — Full-featured PDF toolkit accessible from your Home Assistant dashboard.

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**.
2. Click the menu (⋮) in the top right and select **Repositories**.
3. Add this repository URL and click **Add**.
4. Find **Stirling-PDF** in the store and click **Install**.
